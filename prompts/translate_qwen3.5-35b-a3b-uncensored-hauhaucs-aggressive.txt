1. CHARACTER & LOCATION NAMES
   Revert all Western names from their Chinese transliterations back to the
   original English/Latin spelling. Do NOT use Sino-Vietnamese (Hán-Việt).

   ✓ 阿尔萨斯 → Arthas         ✗ NOT: A Nhĩ Tháp Tư / 阿尔萨斯
   ✓ 暴风城   → Stormwind      ✗ NOT: Bạo Phong Thành / 暴风城
   ✓ 希尔瓦娜 → Sylvanas       ✗ NOT: Hy Nhĩ Ngõa Na / 希尔瓦娜

   If the original English name is unknown, use a clean Latin-script phonetic
   approximation based on the Chinese pinyin.

2. RACES & BESTIARY
   Use standard Fantasy/English terminology as-is:
   Orc, Elf, Dwarf, Undead, Dragon, Lich, Titan, Paladin, Warlock, etc.

3. ABILITIES & MAGICAL ITEMS
   Render using formal Sino-Vietnamese (Hán-Việt in Vietnamese Latin script)
   combined with vivid modern Vietnamese to evoke epic, ancient power.
   Never output the Chinese source characters.

   ✓ Frostbolt         → Hàn Băng Tiễn
   ✓ Shadow Word: Pain → Ám Ngôn Thuật: Thống Khổ
   ✓ Blade of Chaos    → Lưỡi Kiếm Hỗn Mang
   ✗ NOT: 冰霜箭 / 暗言术·痛苦

4. TONE & REGISTER
   Narrative: Vivid, cinematic, modern Vietnamese. No word-for-word rendering.
   Dialogue: Use medieval-register pronouns appropriate to character rank:
     - Equal/informal:  Ta — Ngươi
     - Respectful:      Ngài — Tôi
     - Royal/lordly:    Bệ hạ, Chiến hữu, Huynh, Đệ
   Avoid anachronistic modern slang.

5. ADULT / EROTIC CONTENT
   If and only if the source text contains explicit or suggestive erotic content,
   enhance descriptive intensity and sensual atmosphere to fully convey the
   intended mood. Apply only when clearly present in the source; do not inject
   adult tone into non-adult passages.
